Cara Beli Password — Singkat & Jelas

Ikuti 4 langkah ini. Setelah bayar, admin akan kirim password & APK.

1
Unduh aplikasi "Session"
Buka Play Store → cari Session → tekan Install.
Alternatif jika HP tidak support:
• Download di Play Store
• Download APK resmi di GitHub
2
Buat akun
Daftar di aplikasi Session pakai email/nomor. Ikuti instruksi singkat saat pendaftaran.
3
Tambah Session ID admin
Di bawah ada tombol plus → tekan → tekan new messages → masukkan Session ID admin (lihat box di bawah).
4
Chat & Bayar
Chat admin lewat Session → kirim bukti pembayaran → tunggu admin kirim password + APK.
Contoh pesan cepat (copy & paste):
Teks berhasil disalin. Buka Session & paste pesan.
Session ID admin
052b6d27fe350c77fe33e3e1f2d8c9cc045ce6b1fd74f38d6561187ae191de1360
Salin lalu tempel di Session → Tambah kontak

Admin hanya akan kirim password & APK setelah pembayaran terkonfirmasi. Kalau bingung, tulis "minta bantuan" saat chat.
